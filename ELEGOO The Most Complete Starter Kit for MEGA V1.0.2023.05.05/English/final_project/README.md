PROJECT DESCRIPTION:
	Make a smart deadbolt that include the following accessories/functionality:
		deadbolt ctrl:
			servo
		access   ctrl:
			remote
				-AND-
			keypad
		power:
			9V battery
		display
			lcd display
				-AND-
			serial monitor

COMBINATION CODE:
	022305		 

USAGE GUIDE:
	keypad:
		digits -> password control
		'#'    -> enter password attempt
		'*'	   -> delete password attempt
		'A'	   -> lock
	remote:
		digits -> password control
		UP     -> enter password attempt
		DOWN   -> delete password attempt
		PAUSE  -> toggle lock

PIN ASSIGNMENTS:
	Kindly refer to code for pin assignments
